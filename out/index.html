Frequently Asked Questions

Is this tool really free?

Yes, completely free forever. No signup, no payment, no tricks.

Are my files safe? Do you upload them anywhere?

No. All processing happens entirely inside your browser locally. Files never leave your device.

How many files can I rename at once?

There is no limit. You can rename 1 or 10,000 files at once seamlessly.

Does it work on Mac and Windows?

Yes. It works on any operating system running a modern browser — Chrome, Safari, Firefox, Edge.

Can I preview before downloading?

Yes. Our live preview lets you see the before and after for every file before creating the ZIP.

What file types does it support?

All file types! Images, videos, documents, code, audio — any file type is supported.

Do I need to create an account?

No. There is no signup, no email request, and no account wall.

Can I use regex for find and replace?

Yes. Simply toggle regex mode in the Find & Replace field.
